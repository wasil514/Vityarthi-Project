## Overview  
This project is a simple CLI-based Health Management System built in Python. It allows a trainer ("Master") to log workout and diet plans for multiple clients, and enables clients to view their personalized plans.

## Features  
- Trainer mode:  
  - Log workout plans for clients  
  - Log diet plans for clients  
- Client mode:  
  - Retrieve and view logged workout plan  
  - Retrieve and view logged diet plan  
- Each entry is timestamped for tracking purposes  
- Separate log files for each client and plan type (e.g., `Robin_workout.txt`, `Cadence_diet.txt`)

## Technology Stack  
- Python 3.x  
- Built-in `datetime` module  
- File I/O for data persistence  
- Command-Line Interface (CLI)
